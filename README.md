# Whitaker-Data-Science-Portfolio

I am currently a junior at the University of Notre Dame studying political science, global affairs, and data science. I strive to utilize my quantitative skills from my data science instruction to contribute to research in the field of international relations. I hope to further my skills in data analysis and develop a greater understanding of machine learning techniques. 

This page is a culimination of the projects that have helped me strive towards this goal from my Introduction to Data Science Course. I plan to organize this repository with folders for each project broken down into subsections for code, results, data visualization, and analysis, in a way that makes a project accessible and easily understood from a reader's perspective.

The first project that I conducted was a study of alcohol consumption across countries using unique visualizations in a streamlit application. 

Link to first project: https://github.com/ashbywhitaker/Whitaker-Data-Science-Portfolio/tree/main/basic_streamlit_app 

The second project that I completed was a tidy data project. I studied medal counts across sports and gender in the Olympics after cleaning data, separating conjoined variables, and finally creating visualizations and pivot tables that indicate these changes. This complements my overall portfolio by indicating the skills I have developed in data wrangling. Because tidying and cleaning data is so important to data science, an indication of those skills is important for my overall portfolio.

Link to second project: https://github.com/ashbywhitaker/Whitaker-Data-Science-Portfolio/tree/main/Portfolio_Update_2 

The third project that I completed was a Machine Learning Project. Using VS code and Streamlit, I created an app that allows a user to upload any dataset, or use the sample datasets provided, to run a logistic regression or decision tree model. I created sliders that allow the user to control many aspects of the data including test size and iterations of running. Beyond this, I include discussions of the sample datasets and basic information about the metrics shown. Finally, I include a measure of accuracy and ROC Curve. While the ROC Curve seems to struggle with the multi class element of datasets, the accuracy measure is affected by changing the sliders. Overall, this project adds to my data science portfolio generally as it shows the development of my knoweldge and skills in machine learning. It shows an understanding of the measures used in logistic regression and decision tree models. 

Link to third project: https://github.com/ashbywhitaker/Whitaker-Data-Science-Portfolio/tree/main/MLStreamlitApp
