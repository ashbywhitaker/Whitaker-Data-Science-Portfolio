# Whitaker-Data-Science-Portfolio

I am currently a junior at the University of Notre Dame studying political science, global affairs, and data science. I strive to utilize my quantitative skills from my data science instruction to contribute to research in the field of international relations. I hope to further my skills in data analysis and develop a greater understanding of machine learning techniques. 

This page is a culimination of the projects that have helped me strive towards this goal from my Introduction to Data Science Course. I plan to organize this repository with folders for each project broken down into subsections for code, results, data visualization, and analysis, in a way that makes a project accessible and easily understood from a reader's perspective.

The first project that I conducted was a study of alcohol consumption across countries using unique visualizations in a streamlit application. 

Link to first project: https://github.com/ashbywhitaker/Whitaker-Data-Science-Portfolio/tree/main/basic_streamlit_app
