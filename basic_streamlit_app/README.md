# Drunk by Country – Interactive Data Exploration App

## Project Overview
This project is an interactive data exploration app built using Streamlit in VS Code. The goal of the app is to analyze global alcohol consumption by country and provide visual insights into drinking patterns around the world. Users can explore the dataset through maps, graphs, and summary statistics to better understand how alcohol consumption varies by region and country. 

## Instructions

### Running Locally
1. **Download Necessary Packages**
    pip install streamlit pandas pydeck
2. **Run the Application**
    streamlit run app.py

## App Features

### Datasets
- Uses a dataset containing alcohol consumption statistics by country
- Includes information such as: Beer servings, Wine servings, Spirit servings, Total litres of pure alcohol consumed

### Visualizations
- Data Tab: Allows users to see the raw data being used in the analysis
- Map Visualization: Interactive world map created using PyDeck
- Graphs: Interactive bar charts
- Summary Statistics: Displays statistical summaries of the dataset

## Key Insights
- Europe shows significantly higher alcohol consumption compared to other regions
- Different regions show varying preferences for types of alcohol (beer, wine, spirits)
- Geographic visualization highlights global patterns in consumption
